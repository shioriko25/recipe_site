# <お菓子作りレシピサイト>

## サイト概要
### サイトテーマ
*お菓子に関する自分のレシピを投稿できたり、作りたいお菓子のジャンルを検索して閲覧できるコミュニティサイト

### テーマを選んだ理由
*お菓子作りが趣味でよく利用していて、投稿レシピに「実際に作ってみました」などコミュニケーションが生まれたり、様々なレシピがあるので作りたいレシピがあると便利だと考えました。

### ターゲットユーザ
お菓子を作りたい人

### 主な利用シーン
作りたいお菓子がありレシピを検索するとき、自分のお菓子のレシピを投稿する時
